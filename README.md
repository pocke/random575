# Random 575

Generate 575 randomly.


## Development

```bash
$ git clone https://github.com/pocke/random575
$ cd random575
$ bundle install

# Build JavaScript
$ bundle exec rake

# Build JavaScript and watch the updates
$ bundle exec rake watch
```
